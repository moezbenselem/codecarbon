# Codecarbon

Frontend for the Nuit de l'info challenge (https://www.nuitdelinfo.com/inscription/defis/287)

## Installation

clone or Download the code

Use the package manager npm to install packages.

```bash
npm install
```

## Usage

Launch develepment server

```bash
ng serve
````
